# Project Overview: Hybrid HPC & Big Data Cluster
This repository contains the configuration and scripts for a distributed bioinformatics machine learning cluster.

## Environment Details
- **Architecture:** 3-node cluster (1 Master, 2 Workers) on VirtualBox.
- **Networking:** Static IP assignment (192.168.56.x) with custom netplan configuration.
- **Task 1:** Traditional HPC setup using OpenMPI and Scikit-Learn.
- **Task 2:** Big Data orchestration using Docker Swarm and Apache Spark.
